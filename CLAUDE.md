## 第六部分：项目技术背景

### 6.1 主程序架构

`ThermalAPP_v8_80.m` 是一个单体 MATLAB GUI 应用，所有功能以内嵌子函数的形式组织。核心模块如下：

| 模块 | 关键子函数 | 功能说明 |
|------|-----------|---------|
| 数据加载与回放 | `onLoadFile`, `updateFrame`, `onPlayToggle` | 热成像序列加载、逐帧渲染、播放控制 |
| 图像预处理 | `getProcessedFrame` | 去噪（中值/高斯）、CLAHE 增强 |
| 蒸汽校准 | `onVaporConfigChange`, `onAutoCatch`, `addVaporAnchorPoint` | 基于蒸汽参考温度的多点/多层校准 |
| 图像稳像 | `onCalcMotion`, `onSetStabTarget` | 基于模板匹配的运动补偿 |
| 动态 ROI | `onSetLocalAnchor` | 局部特征追踪的动态锚点 |
| 熔池/HAZ 面积分析 | `onCalcArea`, `performSingleFrameCalc`, `plotAreaCurves` | 基于温度阈值的熔池与热影响区面积定量 |
| NDT 深度映射 | `onSetBaseline`, `onGenerateNDTReport`, `onPlotParity` | 无损检测数据与热成像 Z 深度的关联分析 |
| 项目持久化 | `onExportProject`, `onExportBaked` | 完整分析状态与烘焙数据的保存/加载 |

### 6.2 全局状态结构体 S

所有分析状态集中存储于全局结构体 `S` 中，关键子字段：

- `S.matObj` — MAT 文件对象（热成像数据源）
- `S.N` — 总帧数
- `S.frameIdx` — 当前帧索引
- `S.preproc` — 预处理参数（去噪类型、CLAHE 参数）
- `S.vapor` — 蒸汽校准状态（参考温度、锚点表、Gamma 曲线）
- `S.stab` — 稳像状态（位移向量、参考帧）
- `S.area` — 面积分析状态（阈值、形态学参数、历史记录）
- `S.ndt` — NDT 真值映射表
- `S.dynamicAnchor` — 动态锚点状态
- `S.calibMode` — 校准模式（Single / Multi）

### 6.3 关键物理参数

| 参数 | 默认值 | 物理意义 |
|------|--------|---------|
| `T_melt` | 1400 °C | TC4 钛合金固相线温度（熔池判定阈值） |
| `T_htw` | 600 °C | 热影响区（HAZ）判定阈值 |
| `refTemp` | 1500 °C | 蒸汽参考温度（单点校准基准） |
| `fps` | 5 | 默认回放帧率 |
